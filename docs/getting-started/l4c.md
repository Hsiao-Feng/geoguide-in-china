# Looking4Cache (L4C) 使用指南

在 App Store 中搜索 `Looking4Cache`，或使用[本链接](https://apps.apple.com/cn/app/looking4cache-pro/id532870466)以获取该 App。经过一系列条款阅读后，点选 `Add Geocaching User` 登录你的 Geocaching 账号以开始使用。

![App Store 页面截图](./imgs/l4c-1.jpg){ width="50%" }

![登录 Geocaching 账号](./imgs/l4c-2.jpg){ width="50%" }

在主界面，可以点选下方的两个选卡切换内容：

- **列表**（以列表形式展示搜索到的 Cache）
- **地图**（将 Cache 标记在地图上）

在**列表**选卡中，上方最常用的是最左侧的**搜索**按钮，可以根据条件检索 Cache，并呈现在**列表**或**地图**上。

![软件主界面。下方有两个选卡按钮](./imgs/l4c-3.jpg){ width="50%" }

L4C 的使用逻辑是：先使用**搜索**来获取一定区域或条件的 Cache（如你手机定位附近 25 km 内的），下载这些 Cache 的信息，然后展示在**列表**和**地图**中。

由于一开始并未搜索任何 Cache，因此**列表**和**地图**中均无 Cache 展示。

## 搜索 Cache

点选左上角**搜索**按钮，来到搜索对话框。

根据下图所示，此处默认是：

- 根据手机定位**位置**（Coordinates）为中心
- `25` km **半径**（Radius）圆形范围内
- 要获取的 Cache **数量**（Search Count）为 `20` 个。

你可以根据实际需要修改这些信息。按右上角 `Search >` 按钮即可执行查找。

!!! note "合理设置查找数量与半径"

    对于 Cache 数量多又密集的地区，如果你设置的查找半径较大，建议也增加查找数量，否则可能无法获取全部的 Cache。

![搜索界面](./imgs/l4c-4.jpg){ width="50%" }

以坐标查找 Cache 总是要有一个中心坐标。对于设置中心坐标，点按坐标右侧的图标，可供你选择有几种不同的方法：

- **地图中心（Map Center）**：在主界面的地图选卡中，拖动到你想要作为中心的位置。来到搜索界面选择本项即可。
- **当前位置（Current Position）**：以你手机定位的坐标作为中心。
- **复制的坐标（Clipboard）**：读取你复制的坐标自动填入。需要符合类似格式：`N 11° 45.014 E 019° 19.810`。
- **搜索地址（Search for an address）**：直接搜索地名，例如`上海淮海中路地铁站`。

![搜索界面调整坐标的获取模式](./imgs/l4c-5.jpg){ width="50%" }

此外，切换到 **Cache Code** 选卡，亦可以直接输入 Cache 编号精确搜索。

![搜索界面](./imgs/l4c-6.jpg){ width="50%" }

在搜索后，如果有符合条件的 Cache，即会显示在主页面中，通过切换下方的选卡，可以以列表形式或地图形式查看。
